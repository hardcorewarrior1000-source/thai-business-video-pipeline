=== TEMPLATE: English Translation ===

This template defines how OpenCode translates your Thai script to English.
Say "translate [topic] to english" and I follow these rules.

---

## TRANSLATION RULES

LANGUAGE:
- Translate Thai narration to natural English
- Keep the same 2nd person ("you", "your")
- Maintain the same calm, intelligent tone
- No "we" or "I" — always "you"

STYLE:
- Conversational but smart
- Like a TED talk transcript
- Short sentences preserved
- "..." pauses preserved
- Blank lines preserved

ADAPTATIONS:
- Thai cultural references → universal equivalents
- Thai-specific examples → global tech examples
- Keep technical terms in English (they're already English)
- Maintain the hook → reframe → deep dive → twist → mirror → echo structure

FORMAT:
- Same as Thai version — pure narration
- No headers, no bullets, no stage directions
- Word count should match Thai version (±10%)

---

## WHAT CHANGES

| Thai Version | English Version |
|--------------|-----------------|
| Thai language | Natural English |
| Thai cultural refs | Universal/global refs |
| "คุณ" | "you" |
| Thai study references | Same studies, English names |
| Thai examples | Global tech examples |

---

## WHAT STAYS THE SAME

- Same narrative arc
- Same rhythm (short/long sentences)
- Same hook structure
- Same references/studies
- Same length
- Same "..." and blank line pauses

---

## OUTPUT

Saved to: output/[topic-slug]/english/script.txt

Same format as Thai version — pure narration text.
Ready to paste into ElevenLabs English voice.
