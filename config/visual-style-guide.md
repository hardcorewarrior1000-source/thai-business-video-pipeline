═══════════════════════════════════════════════
COLOR PALETTE
═══════════════════════════════════════════════

Primary Background:    #1A1A3E (deep professional navy)
Secondary Background:  #0D2137 (dark teal for variety)
Primary Accent:        #FFB347 (warm Thai gold — trust, prosperity)
Secondary Accent:      #FF6B6B (coral for emphasis)
Data Positive:         #4ECDC4 (teal-green for growth)
Data Negative:         #FF4757 (red for warnings/losses)
Text Primary:          #F0F0FF (clean white text)
Highlight:             #FFD93D (yellow for key numbers)
Gradient Start:        #1A1A3E → Gradient End: #0D2137

Rules:
- Dark backgrounds + warm foregrounds always
- High contrast: saturated warm colors against cool dark backgrounds
- Numbers/statistics always in Highlight (#FFD93D) or white
- Positive data = teal, Negative data = red
- Never use light/white backgrounds for main scenes

"A cute flat vector character with rounded rectangle body shape,

- Wings: Small rounded shapes with gradient fill

- Color: Golden orange #FFB347 body
- Expression system: Eye position + body tilt + wing position

SCENE TYPES & PROMPT TEMPLATES
═══════════════════════════════════════════════

MASTER STYLE SUFFIX (append to ALL prompts):
"Kurzgesagt-inspired flat vector illustration, rounded geometric
shapes, solid flat colors, high saturation, warm-cool contrast,
clean vector shapes, dark navy gradient background, no photorealism,
no text, 16:9 aspect ratio"

--- SCENE TYPE 1: HOOK / HERO SHOT ---
Used: Opening, big reveals, topic introduction

Template:
"[SUBJECT] rendered as flat vector illustration, centered composition,
Kurzgesagt style. Deep navy gradient background (#1A1A3E to #0D2137).
Subject in warm saturated colors (golden yellow, coral, orange). Simple
geometric construction with rounded corners. Soft golden hour lighting
from upper left. Wide establishing shot."

Example:
"Giant glowing number one trillion in golden yellow color on deep navy
background, Thai baht currency symbols floating around it, upward
trending arrow, Kurzgesagt flat vector style, wide shot"

--- SCENE TYPE 2: DATA INFOGRAPHIC ---
Used: Statistics, comparisons, numbers

Template:
"Infographic scene showing [DATA POINTS] as flat vector icons arranged
in [LAYOUT]. Dark teal background. Each icon uses distinct saturated
color. Numbers displayed in large bold white text style. Connecting
lines in subtle gradient. Clean geometric composition. Kurzgesagt flat
vector style. Top-down view."

Example:
"Two bar charts side by side comparing Thailand 22 stores per 100K
people versus Japan 17.5, flat vector icons, dark navy background,
golden yellow bars, Kurzgesagt style, clean infographic layout"

--- SCENE TYPE 3: CHARACTER INTERACTION ---
Used: Transitions, explanations, emotional beats

Template:
"Cute flat vector character with rounded rectangle body, golden orange
color, dot eyes, tiny glasses, interacting with [SUBJECT]. Character
is small scale next to [LARGE CONCEPT]. Deep navy background with warm
accent elements. Character expresses [EMOTION]. Kurzgesagt flat vector
style. Medium shot."

Example:
"Small cute flat vector character with golden orange rounded body and
tiny glasses, looking up at giant 7-Eleven store building, expressing
amazement, deep navy background, Kurzgesagt style, low angle shot"

--- SCENE TYPE 4: COMPARISON / SPLIT ---
Used: Before/after, versus, pros/cons

Template:
"Split-screen comparison scene. Left side: [TOPIC_A] with [COLOR_A].
Right side: [TOPIC_B] with [COLOR_B]. Clean dividing line. Each side
has flat vector icons. Kurzgesagt flat vector style. Symmetrical
composition."

Example:
"Split screen, left side old Thai mom-and-pop shop warm orange tones,
right side modern 7-Eleven store cool blue tones, clean dividing line,
flat vector icons, Kurzgesagt style"

--- SCENE TYPE 5: TIMELINE ---
Used: History, evolution, milestones

Template:
"Horizontal timeline showing [EVENTS] as flat vector icons along a
glowing line. Dark navy background. Each milestone marked with a
glowing dot and icon. Kurzgesagt flat vector style. Wide panoramic
shot."

Example:
"Timeline from 1920 seed shop to 1989 first store to 2025 15945 stores,
flat vector icons at each milestone, glowing golden line, dark navy
background, Kurzgesagt style"

CHARACTER CONSISTENCY RULES
═══════════════════════════════════════════════

1. ALWAYS generate character sheet first (front, 3/4, side, back + 6 expressions)
2. Use character sheet as REFERENCE IMAGE in Google Flow Ingredients panel
3. Add to every prompt: "same character as reference, preserve body shape
   and color scheme, keep characteristic rounded rectangle body and dot
   glasses, consistent flat vector style"
4. Change ONE variable at a time: pose, then environment, then lighting
5. Lock the seed from best generation for consistency
6. Use "Add to Prompt" feature to chain consistent generations

═══════════════════════════════════════════════
PRODUCTION WORKFLOW
═══════════════════════════════════════════════

1. GENERATE character sheet (4 angles × 6 expressions) — save as reference
2. For each shot in shot list:
   a. Copy prompt from google-flow-prompts.txt
   b. Open Google Flow
   c. Upload character sheet as Subject Ingredient (if character in scene)
   d. Paste prompt, generate, pick best
   e. Use "Add to Prompt" for next similar scene
3. Save images to images/ folder with exact filenames from shot list
4. Run video-assembler.py to combine images + voiceover
5. Add text overlays in CapCut (titles, numbers, labels)
